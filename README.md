# RHAZ

Rhaz is a 2D game created using Unity Game Engine for the purpose of participationg in Global Game Jam 2020. This game is centered around the theme of "Repair". Rhaz takes place in a distant future wear all humanity had been extinguished and a man has accidentally crashed his ship on an unknown planet. He is ambushed with monsters and needs to collect their orbs to repair his ship.

## Getting Started

Our game is deployed on https://simmer.io/@Dazen/rhaz


These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

If you are interested in making edits or running the game on your local machine. You must download Unity Game Engine and clone the file. Once the file is cloned, just open Unity and launch this file.


## Deployment

This game is deployed using simmer.io which is a website for indie game developers to share their games with web browsers. Once the game had been complete, you must export using Unity and it will output a WebGL file which is then used to be uploaded onto simmer.io.
